Loop MCP

Anslut Claude eller ChatGPT till LoopDesks databas med bolagshändelser, finansdata, kungörelser och nyheter.

Online
Server URL
https://loop-mcp-server-production.up.railway.app/mcp
Kom igång
1
Öppna Claude
Gå till claude.ai eller öppna Claude Desktop-appen.
2
Lägg till MCP-server
Claude.ai: Gå till Settings > Connectors > Add connector. Klistra in server-URL:en ovan.

Claude Desktop: Redigera claude_desktop_config.json och lägg till: 
{
  "mcpServers": {
    "loop": {
      "url": "https://loop-mcp-server-production.up.railway.app/mcp",
      "headers": {
        "Authorization": "Bearer DIN_API_NYCKEL"
      }
    }
  }
}
3
Börja chatta
Fråga Claude om svenska bolagshändelser, t.ex:
"Vilka bolag har haft nyemission senaste månaden?"
"Visa mig finansdata för Einride"
"Sök efter konkurser i Stockholm"
1
Öppna ChatGPT
Gå till chat.openai.com (kräver Plus, Team eller Enterprise).
2
Lägg till MCP-server
Gå till Settings > Tools & Connectors > Add MCP Server.

Klistra in server-URL:en:
https://loop-mcp-server-production.up.railway.app/mcp

Lägg till Authorization-headern med din API-nyckel.
3
Börja chatta
Ställ frågor om bolagshändelser, precis som med Claude. ChatGPT kommer använda Loop-verktygen automatiskt.
1
Öppna Cursor Settings
Gå till Cursor > Settings > MCP Servers.
2
Lägg till server
Klicka "Add MCP Server" och klistra in:
{
  "loop": {
    "url": "https://loop-mcp-server-production.up.railway.app/mcp",
    "headers": {
      "Authorization": "Bearer DIN_API_NYCKEL"
    }
  }
}
3
Använd i Agent mode
Loop-verktygen dyker upp automatiskt i Cursors Agent mode. Ställ frågor om bolag direkt i editorn.
